### 2. 世界の公開について
書籍では、つくった世界を「公開」にするように書かれています。

ですが、2026年中旬よりRobloxの規約が変わり、公開には以下が必要になりました。

#### 16歳以上への公開
 - 顔写真を撮影して年齢確認する
 - 公開する世界の内容についてのアンケートに答える


#### すべての人への公開
 - Robloxでいっぱい課金している人に、2ヶ月以内で500回遊んでもらう
 - Roblox Plusを2ヶ月課金、または1000 Robux（90日で返金）を払う
 - マイナンバーカードで本人確認する

設定ができない人は、より簡単な「友達」に公開するように読み替えて進めてください。
