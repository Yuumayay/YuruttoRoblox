## 世界の公開について
書籍の解説では、つくった世界を「公開」にするように書かれています。

ですが、2026年中旬よりRobloxの規約が変わり、公開には以下の設定が必要になりました。

### 16歳以上への公開
 - 顔写真の撮影による年齢確認
 - 公開する世界の内容に関するアンケート
### すべての人への公開
 - Robloxでいっぱい課金している人に、2ヶ月以内で500回遊んでもらう
 - Roblox Plusを2ヶ月課金、または1000 Robux（90日で返金）を払う
 - マイナンバーカードで本人確認

設定ができない人は「友達」に公開するように読み替えて進めてください。
